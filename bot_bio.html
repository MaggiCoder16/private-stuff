FAQ

Common questions.

How do I set up moderation?
Run /setup-moderation in your server. You need to be the server owner or have Ban/Kick/Timeout permissions. A 6-step wizard guides you through AutoMod, log channels, mod roles, link policy, AntiSpam, and AntiRaid. Takes about 2 minutes.
What permissions does Codunot need for moderation?
Exactly 5 permissions: Ban Members, Kick Members, Moderate Members, Manage Messages, and Manage Channels. Go to Server Settings → Roles → Codunot to enable them. The setup wizard will also tell you if anything is missing.
I already had Codunot — do I need to do anything?
Just grant the 5 permissions (Server Settings → Roles → Codunot), then run /setup-moderation once to configure moderation. All existing chatbot features work exactly the same as before.
How do I vote to unlock features?
Visit top.gg/bot/1435987186502733878/vote and click the vote button. Unlocks all vote-locked features for 12 hours. Completely free, takes 10 seconds.
What's the difference between vote-locked and premium features?
Voting (free, every 12 hours) unlocks image generation, music, file reading, TTS, and transcription. Premium/Gold ($10–15 per 2 months) removes daily limits and unlocks advanced mod commands like /tempban, /massban, /modstats, and /note.
Can I use Codunot in DMs?
Yes! In DMs you don't need to mention the bot — just type normally. Moderation commands only work in servers.
How does code testing work?
Use /test_code to open a modal, paste your Python code, and Codunot runs it in a secure sandbox. Real execution results, not AI guesses. If there's an error, the AI automatically fixes the code and re-tests it.
